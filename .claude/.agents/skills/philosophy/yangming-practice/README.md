# 王阳明心学实践

**用王阳明心学做现实决策与行动校准**

## 概述

这是一个基于 **王阳明心学**（心即理、致良知、知行合一、事上磨炼、破心中贼）的实践框架。帮助用户在现实问题中做到：澄清良知、识别心贼、形成行动、当日验证。

## 核心价值

- **向内找答案**：先回到内在动机与原则，再谈外部策略
- **减少自我欺骗**：识别拖延、合理化、面子、恐惧等“心中贼”
- **行动导向**：任何结论必须转化为具体可执行行动
- **当日验证**：在具体事务中修正认知，而非空谈

## 使用方式

1. 在 Claude 对话中复制 [[SKILL.md]] 中的完整 Prompt
2. 回答 Claude 提问，聚焦你的真实问题
3. 获得 5 段式结构输出：
   - 事（当前关节点）
   - 心（良知判断）  
   - 贼（主要内在阻力）
   - 行（24小时行动计划）
   - 省（复盘问题）
